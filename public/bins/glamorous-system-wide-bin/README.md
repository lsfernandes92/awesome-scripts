## >_Short description
Make individual bin executable system-wide.

</br>
</br>
</br>
</br>

## >_Description
This means the binary will be added to `/usr/local/bin`, which is in your
`$PATH` by default.

Anything in your `$PATH` is available from anywhere in your system.

</br>
</br>
</br>
</br>
